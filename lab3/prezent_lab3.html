Цель работы

Получить практические навыки работы в Linux с правами доступа для групп пользователей и определить минимальные права, необходимые для выполнения типовых операций в общей директории.

2

Создание пользователей и группы

useradd guest passwd guest useradd guest2 passwd guest2 gpasswd -a guest2 guest
  • созданы пользователи guest и guest2
  • guest2 добавлен в группу guest
3

Проверка пользователей и групп

pwd whoami groups guest groups guest2 id -Gn id -G cat /etc/group | grep guest newgrp guest
4

Изменение прав каталога

chmod g+rwx /home/guest chmod 000 dir1
  • при правах 000 пользователь guest2 не может войти в каталог
  • права директории влияют на доступ к объектам внутри неё
